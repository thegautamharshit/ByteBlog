# ByteBlog

ByteBlog is a modern, minimalistic blog platform designed for smooth writing and reading experiences. It features a rich text editor, robust authentication, and a scalable backend. ByteBlog ensures performance and security while keeping the user experience simple and elegant.

## 🌐 Live Demo

Check out ByteBlog here: [ByteBlog on Vercel](https://byteblog-dusky.vercel.app/signin)

## 🚀 Features

- 📝 **Rich Text Editing** using TipTap
- 🔐 **JWT Authentication** for secure login & registration
- 📚 **Cloudflare Workers** as the backend server
- 💾 **PostgreSQL with Prisma ORM** for efficient data management
- ⚡ **TypeScript for type safety** across frontend and backend
- 🎯 **Zod Validation** for input handling and type inference
- 🚀 **Deployed on Vercel** for fast and reliable hosting

## 🏗 Tech Stack

- **Frontend:** React, TypeScript, TipTap (Rich Text Editor)
- **Backend:** Cloudflare Workers, TypeScript, JWT Authentication
- **Database:** PostgreSQL with Prisma ORM (connection pooling)
- **Validation:** Zod for schema validation and type inference
